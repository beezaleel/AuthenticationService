# AuthenticationService

### MySQL scripts
1. The name of the script is `auth.sql`
2. Copy and run from `MySQL WorkBench`

### How to build
1. Please note that this project only runs in release mode because of mysql connector.
2. Right click on solution and click build.
3. Start services in this order, `AuthenticationService`, `ChatServer` and `ChatClient`
4. Right click on `AuthenticationService` and run
5. Right click on `ChatServer` and run
6. Right click on `ChatClient` and run
